Directrices Fotográficas

✅ Hacer

  • Usar luz natural cuando sea posible
  • Mantener consistencia en fondos
  • Mostrar productos desde múltiples ángulos
  • Incluir referencias de escala
  • Optimizar para web (WebP/JPEG)

❌ Evitar

  • Fondos distractivos o complejos
  • Imágenes pixeladas o borrosas
  • Sobreexposición o subexposición
  • Marcas de agua visibles
  • Archivos muy pesados (>2MB)

📋 Guía de Implementación

✅ Mejores Prácticas

  • Consistencia: Usar siempre las variables CSS definidas
  • Contraste: Verificar ratio mínimo 4.5:1 para texto
  • Espaciado: Seguir el sistema de múltiplos de 8px
  • Responsive: Probar en todos los breakpoints
  • Performance: Optimizar gradientes para dispositivos móviles
  • Accesibilidad: Incluir estados focus visibles

⚠️ Evitar

  • Usar colores hardcodeados en lugar de variables
  • Mezclar más de 3 colores en un mismo componente
  • Aplicar gradientes en áreas de texto extenso
  • Usar pesos de fuente no definidos en el sistema
  • Ignorar los estados hover y focus en elementos interactivos
